# Custom Browser Tab

A Foundry VTT module to customize your browser tab title, favicon, and logo icon.

## Features

- **Page Title** — Set a custom browser tab title.
- **Active Scene in Title** — Optionally display the active scene name before, after, or instead of the custom title.
- **Favicon** — Replace the browser tab icon with any image.
- **Logo Icon** — Replace the Foundry anvil logo with a custom image.

## Installation

1. In Foundry VTT, go to **Add-on Modules** and click **Install Module**.
2. Paste this manifest URL:
   ```
   https://raw.githubusercontent.com/cvaliente/Custom-Browser-Tab/master/module.json
   ```
3. Click **Install**.

## Compatibility

- Foundry VTT v10+
- Verified on Foundry VTT v13
